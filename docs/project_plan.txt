1. Objective

Build a mobile-first Progressive Web App (PWA) for daily logging.

The app should:

Be installable on mobile
Be extremely fast to use (tap-first UX)
Allow partial logging throughout the day
Work offline and sync later
Use Google Sheets as the database

Primary goal:

Minimize friction of daily logging while capturing structured data for future insights.

2. Tech Stack
Frontend

IndexedDB (for offline storage)
Backend
Google Apps Script (as REST API layer)
Google Sheets as database
Auth
Google Sign-In (OAuth)
3. Deployment Strategy
Preferred Approach

Backend as Google Apps Script Web App

Reason:
